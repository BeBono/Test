Pasos Git para crear proyecto:

1. dir: para mostrar los archivos que contiene la página web en la carpeta. Antes, el promp debe indicar que está en ese carpeta (abrir carpeta clic derecho GIT Dssh)
2. git init: para crear repositorio local.
3. git status: para comprobar que los archivos aun no están presentados en el escenario.
4. git add .: agrega todos los archivos de la carpeta para que esté pendiente de los cambios. Paso previo al commit.
5. git commit -m "Text from changed": Salva la versión actual de los archvos (screemshot).
6. git status: Para verificar el guardado o commit.
7. git checkout -- .: Para recuperar la última versión guardada en el repositorio.
